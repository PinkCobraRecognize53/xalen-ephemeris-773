# XALEN Enterprise

**XALEN Ephemeris is free and open source under Apache-2.0 — and always will be.**
Everything you need to compute charts is in this repository, forever, at no cost.
Self-host it, embed it, ship it commercially: the license already allows all of it.

**XALEN Enterprise** is for organizations running XALEN in production that need
guarantees, premium capabilities, and a vendor relationship an open-source
project alone cannot provide. It never restricts the open-source core — it adds
to it.

---

## What Enterprise adds

### Assurance
- **IP indemnification & warranty** — Apache-2.0 is provided *"AS IS"* with no
  warranty. An Enterprise agreement adds a commercial warranty and IP
  indemnification, so your legal team can adopt XALEN with confidence.
- **SLA-backed support** — guaranteed response and resolution times, a named
  engineer, private security disclosure, and a direct channel — not just public
  issues.
- **Long-term version support** — backported fixes and extended maintenance for
  the versions you depend on.

### Hosted & managed
- **XALEN Cloud API** — the same engine as a managed, scaled, high-availability
  HTTP API, so you never operate ephemeris infrastructure yourself.
- **Hosted high-precision kernels** — managed DE440/DE441 access for
  sub-arcsecond results without hosting large kernel files yourself.

### Premium capabilities
- Certified-accuracy & compliance reports — auditable cross-validation against
  Swiss Ephemeris / JPL.
- High-throughput batch & bulk computation.
- White-label and on-premise deployment support.
- Advanced and custom predictive modules built to your specification.
